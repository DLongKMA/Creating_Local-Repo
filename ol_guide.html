Hướng dẫn cài đặt repo cho Oracle Linux
  • Tải file .repo tại đây:
    • Oracle Linux 6 (i386)
    • Oracle Linux 6 (x86_64)
    • Oracle Linux 6 (sparc64)
    • Oracle Linux 7 (x86_64)
    • Oracle Linux 6.9 (x86_64)
    • Oracle Linux 8 (x86_64)
  • Đặt file vào folder /etc/yum.repos.d/
  • # yum update để update thông tin repo
